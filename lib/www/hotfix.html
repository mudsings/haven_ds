Dead Souls - Hotfixes

Nothing's prefect. This is most certainly true of anything I've touched.

This page contains the late-breaking bugfixes found between Dead Souls releases. Only important bugs are addressed here. Feature enhancements and minor bugfixes become available on full release.

All of the fixes listed here will be included in the next release. They are provided here early in order to address potential problems.

For Dead Souls 2.0r22, the following fixes are available:

  • snooper bug: Right before releases, I comment out debug statements. Unfortunately, this one spanned more than one line. Replace your existing /secure/obj/snooper.c with this one.
  • stargate bug: The stargate system underwent a major revamp, and a couple of bugs snuck in. When a non-existent gate is dialed, it now fails ungracefully. Replace /lib/stargate.c with this file and replace /daemon/stargate.c with this file.
  • channel bug: It appears that r22 introduced a bug that sends local channels to the i3 router. I wasn't able to reproduce this with an r21 mud, but that isn't definitive. In the meantime, rest assured I have not been snooping on anyone, and please copy this file over your current /daemon/services/channel.c then: update /daemon/services/channel.c and then: update /daemon/services.c . Please accept my apologies for the appearance of privacy intrusion.
