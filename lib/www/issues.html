Known Issues

    The following are the known issues, bugs, or misfeatures that exist in Dead Souls 2.1. There is no timeframe for their correction, but unless otherwise noted here, they will eventually be addressed.

  • Channel messages are sometimes entered twice into the channel logs.
  • The "gauge" command (estimates CPU processing time) does not work on Windows because of an incompatible system call. No fix planned.
  • If germs hang around a long time in a room, they can cause errors when they try to infect someone.
  • The blackjack table contains some calculation errors. No fix planned.
  • The Windows addr_server.exe program caused the driver to fail, and its functionality is of such limited helpfulness that it is no longer included. No fix planned.
  • Having customdefs included in files under the /domains subdir makes QCS misbehave.
  • A peculiarity in MudOS grammar can be seen when a room contains more than one corpse.
  • Combat messages sometimes appear out of order.
